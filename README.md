# handy

> yet another alfred-like application, simple but really useful, built with electron


## Dev

```
$ npm install
```

### Run

```
$ npm run dev
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
